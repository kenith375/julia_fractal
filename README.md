# julia_fractal
A modified Julia Fractal shader.

See it in action:


How to get it yourself:
  Download source code obviously,
  install the following dependecies:
    python,
    pygame,
    moderngl
  Run main.py and watch as it throws an error because you didn't have all the dependencies.
  If it did work, you can see the fracatl render in real time.


Their are some parameters you can change in the main.py file and if you remove the hashes near the end, it saves all the frames.
(You must create a folder called frames in the same folder if you want to save it. Also saving is slow)
